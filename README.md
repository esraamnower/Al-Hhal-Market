# Desktop Accounting System

A Flutter desktop application for Windows that provides a complete offline accounting solution.

## Features

- Authentication & Security
- Sales Management
- Purchase Management
- Invoice Management
- Supplier Invoices
- Receipts
- Cash Boxes
- Customer & Supplier Management
- Inventory Management
- PDF Export
- SQLite Local Database
- Device Activation & Licensing
- Backup & Restore
- Daily Transactions
- Dashboard & Account Summary
